# 出差费用管理规定

## 城市分类

出差目的地按城市经济水平分为A、B、C三类：

| 城市类别 | 包含城市 |
|---------|---------|
| A类城市 | 北京、上海、广州、深圳 |
| B类城市 | 杭州、南京、成都、武汉、西安、天津、重庆、苏州 |
| C类城市 | 其他城市（未在A类、B类中列出的所有城市均归C类） |

## 住宿费报销标准

员工出差住宿费按目的地城市类别执行以下标准，超出部分由员工自行承担：

| 城市类别 | 住宿费上限（元/天） |
|---------|----------------|
| A类城市 | 500 |
| B类城市 | 400 |
| C类城市 | 300 |

## 餐饮补贴标准

| 城市类别 | 餐补（元/天） |
|---------|------------|
| A类城市 | 100 |
| B类城市 | 80 |
| C类城市 | 60 |

## 交通费规定

- 出差距离500公里以上可乘坐飞机经济舱
- 出差距离200公里至500公里可乘坐高铁二等座
- 市内交通凭票报销，单次不超过100元

## 报销流程

1. 出差前：提交出差申请，填写目的地、时间、预计费用
2. 出差中：保留所有票据（住宿发票、交通票据、餐饮收据）
3. 出差后5个工作日内：提交报销申请，附上票据原件
4. 直属上级审批后，财务在3个工作日内完成打款
